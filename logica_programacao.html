Requisitos

Os únicos conhecimentos necessários são boa vontade e curiosidade em aprender a lógica dos computadores e a lógica racional para começar a escrever os primeiros programas. É preciso saber usar o Windows somente.

Descrição

Neste curso o aluno aprenderá os seguintes tópicos:

  • Conceitos Iniciais para a área de programação (operadores, algoritmos, linguagens, etc);
  • Conhecerá a linguagem de programação e/ou representação de algoritmos VISUALG;
  • Aprenderá a representar qualquer tipo de programa de computador por meio de algoritmos utilizando as estruturas existentes em todas as linguagens de programação;
  • Estruturas de Seleção
  • Estruturas de Repetição
  • Vetores
  • Matrizes
  • Testes de software
  • Criar Funções
  • Converter algoritmos em outras linguagens
  • Diferenciar Erro lógico de Erro de Sintaxe
  • Resolver problemas por meio de algoritmos

Para quem é este curso:

Iniciantes que queiram aprender a programar em qualquer linguagem de programação, pois a lógica é a base e o software VisuALG é o software para ver a sua lógica sendo executada, assim como acontece nas linguagens de mais alto nível.
